# Mikroupravljači — ATmega32U4

Materijali za laboratorijske vježbe kolegija **Mikroupravljači** na Visokoj školi za informacijske tehnologije (VUB), uključujući skeletni kod vježbi, potpuna rješenja i udžbenik.

## Sadržaj

| Modul | Vježbe |
|-------|--------|
| Digital Output | vjezba411 – vjezba418 |
| Digital Input | vjezba421 – vjezba426 |
| LCD display | vjezba511 – vjezba518 |
| EEPROM memorija | vjezba611 |
| ADC | vjezba711 – vjezba715 |
| Tajmeri | vjezba811 – vjezba816 |
| PWM | vjezba821 – vjezba826 |
| SSD-SBR | vjezba911 – vjezba915 |
| UART | vjezba1011 – vjezba1015 |
| Vanjski prekidi | vjezba1111 – vjezba1112 |
| Moduli (integrirane vježbe) | vjezba1211 – vjezba1261 |

Svaki modul sadrži dvije mape:
- **`vjezbe/`** — skeletni kod za izradu vježbi
- **`rjesenja/`** — potpuna rješenja

## Hardver

- Mikroupravljač: **ATmega32U4** (Arduino Leonardo kompatibilan)
- IDE: **Atmel Studio**
- Jezik: **C++** (AVR dialekt)
